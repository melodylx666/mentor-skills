# 最佳实践指南

> **版本**：完整版（Full）— 适用于技术选型/参数调优/性能优化/最佳实践类问题
> **使用场景**：用户询问"最佳实践"、"怎么做"、"如何优化"、"方案对比"、"技术选型"
> **输出模式**：支持 Full（完整版）和 Lite（精简版）两种模式
> **信源规范**：每个章节的每条结论/推荐必须标注可追溯的 **信源链接 + 可信度等级**，遵循 [`source-credibility.md`](../references/source-credibility.md) 的分级标准（A 权威 / B 可信 / C 参考 / D 推测）
